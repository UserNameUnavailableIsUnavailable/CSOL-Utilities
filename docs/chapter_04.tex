配置完成后，在罗技控制台中重新保存并运行脚本，以使配置生效。

\subsection{使用方法}

在 0 模式下，将鼠标光标移动到需要购买的物品的兑换按钮处。
随后，切换到 4 模式，购买将自动执行。
切换回 0 模式，购买操作将停止。
